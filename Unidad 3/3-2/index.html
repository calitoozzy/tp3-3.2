3-2 Trabajo practico

Algoritmo es la representación gráfica de la resolución de un problema, mediante diagramas de flujo se van "dibujando" cada paso de un proceso a seguir. Podría decirse que una interpretación semántica.

Ejemplo: Algoritmo Ο = Inicio
◊ = Num A
Num B
[] = Suma = Num A+ Num B
≡ Mostrar suma
Ο = Fin

Pseudocódigo, Es la descripción de cada paso a realizar para llegar a la solución del problema. Es una representación de la sintaxis del detalle/estructura del proceso.

Ejemplo: Pseudocódigo
1.- Algoritmo Sumar;
2.- Var
3.- Entero Num A, Num B, Suma;
4.- Inicio
5.- Escribir('Ingresar Num A y Num B:');
6.- Leer(Num A y Num B);
7.- Suma = Num A + Num B
8.- Escribir('La suma es: Suma');
9.- Fin
